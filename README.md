# 人物
## 伍连德
### 个人简介

* 1879年3月10日－1960年1月21日，马来西亚华侨。
* 1896―1899年留学英国剑桥大学期间获得7个学位。
* 1910年12月东北发生鼠疫大流行，清政府任命伍连德为东三省防鼠疫全权总医官，仅用三个月时间扑灭疫情。
* 首位获诺贝尔奖提名的华人。
